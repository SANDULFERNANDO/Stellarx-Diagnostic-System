# StellarX Diagnostic System

AI-powered point-of-care system for tinea diagnosis in Sri Lanka's Dry Zone

## 🏥 Project Overview

StellarX is an AI-powered diagnostic system designed to assist frontline healthcare workers in accurately detecting tinea (ringworm) infections in Sri Lanka's Dry Zone. The system combines a hybrid Convolutional Neural Network (CNN) with structured clinical symptom data through a fusion layer, delivering enhanced diagnostic specificity over image-only approaches.

## 🎯 Key Features

- ✅ AI-Powered Tinea Detection (EfficientNet CNN)
- ✅ Hybrid Inference (Image + Symptoms Fusion)
- ✅ Differential Diagnosis (2-3 Alternative Conditions)
- ✅ Clinical Decision Support (Pathogen ID, Resistance Risk)
- ✅ Complete Case Management
- ✅ Patient Privacy (Complete Anonymization)
